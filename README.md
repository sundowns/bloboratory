## Controls

| Key        | Action           |
| ------------- |:-------------:|
| mouse1      | toggle obstacle |
| t      | toggle tower placement      |
| s | toggle spawner      |
| g | toggle goal      |
| f1 | toggle debug overlay      |
| f5 | restart game      |
| esc | exit game      |
