## Controls

| Key        | Action           |
| ------------- |:-------------:|
| mouse1      | select/deselect structures |
| 1 | place obstacle |
| 2 | place saw |
| 3 | place cannon |
| 4 | place laser |
| r | rotate selected tower |
| x | refund selected tower |
